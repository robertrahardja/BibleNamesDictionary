# BibleNamesDictionary

Data repository containing a Bible names dictionary in multiple formats.

## Contents

- `BibleNames.js` — JavaScript object form of the dictionary
- `BibleNamesPlist.plist` / `BibleNamesPlist2.plist` — Apple property list formats
- `NSDictData` — archived NSDictionary binary
- `BNtestjs.js` — JS test harness
- `NamesLeftToScrap.txt` — scraping todo list

Source data consumed by the `BibleNamesMacOS` app. Data only — no deployment.

Repo: <https://github.com/robertrahardja/BibleNamesDictionary>
